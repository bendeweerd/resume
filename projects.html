Projects

Calvin University Mars Rover Design Team

I am an active member of Calvin's Mars Rover Team. I designed, modeled, and 3D printed the "wrist" of the rover arm, achieving the 4th and 5th degrees of freedom. I was also responsible for designing and 3D printing parts for other parts of the rover.

Python Chess

I am currently working on creating a game of chess in Python, complete with board, pieces, move logic, and highlighting. The project uses nested dictionaries and other data structures.

Robotic Arm

I am currently designing and building my own robotic arm. While it is a work in progress, I have designed and constructed custom NEMA 17 gearboxes and a slew bearing for the arm's base. I plan to control the arm with a raspberry pi and Arduino Due, which has a higher clock speed that will allow me to smoothly and simultaneously control the five stepper motors.

Luminous Display Pedestal

This was a hybrid woodshop and Arduino/programming project. I constructed the table in the FHS woodshop, then added the LED strips and controller that I programmed at home. The Arduino Sketch for the project contains 398 lines of code, most of which I wrote from scratch.

This Website

Building this website has been an enjoyable challenge, and I've learned a lot doing it. This is the 3rd iteration; the first was pure HTML and the second incorporated some CSS. This version also uses the Bootstrap library to enable responsive behavior for different screen sizes.

Dorm Room RGB Lights

I programmed these lights for my dorm room. They display 8 different effects and are driven by an Arduino. Much of the code was taken from my Luminous Display Pedestal project, with modifications for more effects and modified controls.

Falcon Heavy Model

This mostly 3D-printed model contains individually aggressable RGB LEDs, powered by an Arduino. Masked by the rocket plume, they appear similar to actual rocket exhaust.

Feathered-Leg Table

I constructed this table, which incorporates a unique leg design, in my sophomore year of high school. One picture shows me with my shop teacher Rick Tank and friend Keegan at the MITES state championship.
